# PDF Viewer with Color Inversion & Fullscreen

A simple **Python PDF Viewer** built with **Tkinter** and **PyMuPDF**, featuring:

- Page navigation (next/previous page)
- Ctrl + Mouse wheel zoom in/out
- Color inversion toggle (night mode)
- Fullscreen toggle (button + F11, exit with Escape)
- Centered page display
- Scroll and drag support
